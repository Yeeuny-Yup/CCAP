Implementation Notes (Paper-faithful details)

  • Exact statistic used: for each sample x, we compute channel-wise ||O_{F,C}(x)||_1 by summing absolute activations over all non-channel dimensions, then estimate E[||O_{F,C}(x)||_1] over the calibration set.
  • Unbiased expectation over samples: accumulation is performed by summing ||O_{F,C}(x)||_1 over all samples and dividing by the total number of samples, which remains correct even if the last batch size differs.
  • Zero-masking location: modality-wise zero-masking is applied at the input (Noisy-only / BCM-only), matching the method definition.
  • Numerical stability: ε is used only to avoid division-by-zero when the multimodal reference response is very small; it does not change the intended scoring behavior.
